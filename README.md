luhnch
=====

An OTP library

Prerequisites
-------------

* [Erlang](https://www.erlang.org/downloads)
* [Rebar 3](http://rebar3.org/docs/getting-started/)
* [Rust](https://rustup.rs)

Build
-----

    $ rebar3 compile


Test
----

To run unit tests:

    $ ./test.sh

To run benchmarks:

    $ ./benchmark.escript
